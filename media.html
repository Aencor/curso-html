Imágenes

Las imágenes en HTML se agregan con la etiqueta <img> y tiene los siguientes atributos principales :

  • src : la fuente de la imagen, puede ser local o pública
  • alt : Texto alternativo de la imagen para casos en que no cargue o para screen readers
  • title : Título de la imagen para screen readers
  • description : Descripción de la imagen para screen readers
  • width : Ancho de la imagen
  • height : Alto de la imagen
