Elementos de una etiqueta

Una etiqueta de HTML por lo general tiene una apertura y un cierre

Ej:

Etiquetas Inline

a - referencias e hipervínculos

La etiqueta "a" sirve para poner enlaces como este. Se compone de lo siguiente:

strong - Negritas

La etiqueta strong se usa para agregar negritas

em - Italicas

La etiqueta em se usa para agregar italicas

u - subrayado

La etiqueta u se usa para agregar subrayado

span - comodín

La etiqueta span es usada como un comodín para poder segmentar o contener una parte del texto y agregarle algo después, es muy útil cuando buscas estilizar contenido, por ejemplo este texto va a ser todo en mayúsculas

label - Título de Inputs en formularios

La etiqueta label se utiliza sobretodo en formularios para indicar el título de un input

Etiquetas de Bloque

Son etiquetas que tal como su nombre lo indica crean bloques de contenido y ocupan todo el espacio de la pantalla

div - Crea divisiones como si fueran bloques de Lego y tiene sub etiquetas que sirven de forma semántica:

  • header - sirve para poner bloques de encabezados de contenido
  • footer - sirve para poner bloques de pie de contenido
  • main - sirve para poner bloques de contenido principal
  • aside - sirve para poner bloques de contenido que no son referentes al bloque principal
  • nav - sirve para poner bloques de navegación

ul, ol - Etiquetas de listas

ul - unordered lists

  • Estas listas tienen viñetas
  • para separar elementos

ol - ordered lists

Los elementos ol y ul, llevan sub elemento li que es una etiqueta de linea

  1. Estas listas tienen números
  2. para separar elementos
